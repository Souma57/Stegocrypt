# Stegocrypt 🔐

Stegocrypt is a web-based application that allows users to hide and extract secret messages inside images using steganography techniques.

## Features
- Hide text inside images
- Extract hidden messages
- Simple and user-friendly interface
- Optional encryption for security

## Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- OpenCV / PIL

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/

## Use Cases
- Secure communication
- Data hiding
- Privacy protection
